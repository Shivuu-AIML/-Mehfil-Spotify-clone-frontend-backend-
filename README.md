# Mehfil — Spotify-style frontend for your backend

"Mehfil" is a Hindi/Urdu word for a gathering held for music or poetry — felt like the right name for a
listener + artist platform. A quick vocab pick for you: the app also leans on **convivial** (adjective —
friendly, lively, good-humoured in a social setting), which is the mood the UI is going for.

Mehfil is a full-stack music streaming platform inspired by Spotify, built with modern web technologies. It enables users to create accounts, upload tracks, organize music into albums, and manage their personal music library through a clean, responsive interface. The project focuses on secure authentication, scalable backend architecture, and an intuitive user experience.
Features
User authentication (Sign Up / Login)
Secure session management
Upload and manage music tracks
Create and organize albums
Browse albums and tracks
Responsive, modern UI
RESTful backend API
Database integration for persistent storage
File upload support
Full CRUD operations
